# CoinTrendz Radar

Real-Time Crypto Sentiment & Trending Coins Dashboard

## Features
- Fear & Greed Index gauge (Chart.js)
- Bitcoin & Ethereum price cards with 24h sparklines
- Top 10 gainers and losers by 24h change
- Trending coins from CoinGecko
- Auto-refresh every 60 seconds
- Dark/light mode toggle
- Multi-language support (8 languages)
- Mobile-responsive design
- SEO optimized with JSON-LD, Open Graph, sitemap

## Tech Stack
- HTML/CSS/JS (vanilla)
- Tailwind CSS (CDN)
- Chart.js (CDN)
- CoinGecko API (free)
- Alternative.me Fear & Greed API
- Hosted on Vercel
